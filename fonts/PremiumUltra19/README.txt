This download consists of a three fonts as a compressed, zipped archive:-

Font 1 - A standard 'closed contour' True Type version of the font for use in any Windows application that uses True Type fonts.


Font 2 - An 'open contour' single line version of the font in SVG format, suitable for use with the AxiDraw pen plotter, which supports direct font substitution and mail merge applications, using software available to AxiDraw users. Suitable for other pen plotters that can use open contour SVG fonts.
 
Font 3 - An 'open contour' single line version of the font in True Type format suitable for pen plotters and other CAD equipment which has software designed to use these types of fonts.
