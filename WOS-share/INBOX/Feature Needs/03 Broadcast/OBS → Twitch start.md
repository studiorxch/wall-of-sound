# OBS → Twitch start

Broadcast: Twitch/ OBS 
This feature aims to quick start now stream easily for OBS to Twitch

1. Define:
	1. Playlist
	2. Sampler
	3. Countdown Timer (default: 5 min)
	4. Stream information (If possible)
		1. Title / Description
		2. Notification Message
	5. Social Media notifications (optional)
2. Start Stream
3. Add to scheduler. This is useful when we plan to keep archive of scheduled events (including testing events), also when the scheduler is locked into our website. This is one feature that needs to be integrated into our website.

Once all fields have been filled, start stream button is selected and the entire connection is made. 
- Music starts with default timer.
