This prevents:
- geographic truth collapse
- source absolutism
- location overreach
- environmental canonization

The distinction between:
- Truth
- Mood
- Reference
- Fiction

## Strongest Workflow Advancement

NOT:
- renderer ownership
- world simulation
- cinematic composition tooling
- runtime orchestration

This boundary is extremely important.
