# Sources
## Core Source Categories

### 1. **Brand Systems**

Use these to study how identity becomes repeatable.

Good sources:

- Record labels
- Film studios
- Transit authorities
- Outdoor gear brands
- Broadcast networks
- Museum identities
- Magazine layouts
- Festival posters
- Sports team graphics
- Tech product launches

What to collect:

- Logos
- Typography systems
- Color palettes
- Grid layouts
- Poster families
- Packaging
- Signage
- Motion graphics stills
- UI treatments

Best use for Image Lab:  
**Build repeatable StudioRich visual rules.**
