# Sources

## Pinterest

Pinterest is strongest as a **visual search and expansion tool**.

Use it for:

```
finding similar images
discovering unexpected adjacent styles
collecting visual families
testing visual direction
refining taste
surfacing repeated patterns
finding stronger versions of weak ideas
```

Pinterest is especially useful when you start with a rough phrase like:

```
industrial signage
gold martial arts poster
broadcast HUD graphics
vintage transit typography
monoline warning icons
ambient album cover
```

Then Pinterest helps you visually sharpen the lane.

In Image Lab terms:

```
Pinterest = Discovery + RefinementImage Lab = Interpretation + Conversion + Use
```
