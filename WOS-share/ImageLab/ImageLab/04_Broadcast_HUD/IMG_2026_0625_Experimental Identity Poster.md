# COPY --- Experimental Identity Poster
![[2bbc855512e7574fda2c488fda3653fc.jpg]]
## Visual Summary

A bold poster using oversized geometric typography, strict modular
grids, technical annotation blocks, circular structural forms, and a
restricted red/black/white palette. It presents identity as an
engineered system rather than expressive artwork.

## Core Read

The value of this reference lies in its repeatable design logic:
architectural typography, macro--micro information hierarchy,
disciplined spacing, and documentation aesthetics that imply a larger
operational system.

## Visual Signals

-   Architectural display type

-   High negative space

-   Restrict palettes for maximum clarity.
-   Use circles as structural anchors.
