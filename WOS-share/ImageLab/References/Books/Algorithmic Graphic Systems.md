# Algorithmic Graphic Systems

## Definition
Graphic systems generated from reusable rules rather than handcrafted layouts.

## Core Principles
- Rules before artifacts
- Systems before assets
- Variation with consistency
- Designer as curator

## Reference Sources
- Analog Algorithm
- The Vignelli Canon
- Designing Programmes
- Processing / p5.js
- Transit wayfinding systems

## StudioRich Applications

### WOS
Maps, HUDs, signage, broadcast graphics.

### Image Lab
Extract visual rules from references.

### Brand Identity
Flexible identity systems.
