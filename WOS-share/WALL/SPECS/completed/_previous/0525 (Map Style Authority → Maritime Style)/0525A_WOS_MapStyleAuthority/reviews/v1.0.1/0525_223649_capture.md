2. AtmosphericReadability Authority Boundary
"MapStyleAuthority may NEVER elevate a vessel to a higher visibilityClass than assigned by AtmosphericReadability."

This is the critical separation layer ﾑ Atmosphere owns what is visible (FULL, SILHOUETTE, MARKER_ONLY, ATMOSPHERIC_HIDDEN). Style owns how it looks within that class.

3. Motion Presentation Authority Hardening
ts
type MotionPresentationStyle = {
  readonly headingVisualSmoothing: number
  readonly interpolationCurve: InterpolationCurve
  readonly visualEasingMs: number
}
Critical doctrine:

"MotionPresentationStyle affects ONLY presentation-layer visual easing. It may NEVER alter runtime cadence, fixed timestep accumulation, AIS continuity, dead reckoning, or continuity authority."

4. Presentation Manifest Pattern
"MapStyleAuthority produces immutable presentation manifests consumed by MarineRenderer. MarineRenderer remains authoritative owner of render execution state."

Clean separation between style definition and render execution.

5. Override Governance
"Multiple concurrent override authorities are forbidden. Overrides are ephemeral unless explicitly serialized through SurfaceRuntime tooling."

Prevents override chaos. Live iteration for development, but production overrides require explicit serialization.

6. Projection Integrity Hardening
"Projection-invalid rendering is forbidden. Presentation systems may NOT introduce false spatial depth relationships that contradict runtime spatial truth."

Prevents style system from breaking geographic accuracy.

7. LandStyle Added

8. Wake Presentation Clarification
"wakePresentation parameters affect ONLY visual wake rendering. They may NOT alter WakeRegistry buffers, wake persistence duration, wake decay authority, or wake continuity state."

Preserves WakeAuthority's exclusive ownership of wake memory.
