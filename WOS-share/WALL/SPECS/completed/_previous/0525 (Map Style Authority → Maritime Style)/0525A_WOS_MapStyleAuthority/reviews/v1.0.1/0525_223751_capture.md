# 1. CORE ARCHITECTURAL DOCTRINE

The `MapStyleAuthority` module serves as the centralized, registry-driven configuration engine for the Wall of Sound (WOS) world presentation layer. It transitions the application away from hardcoded rendering drawing routines toward a pure, data-authored model.

- **Presentation systems interpret the world. They do NOT create runtime truth.**
- **2D owns truth. 2.5D owns presentation.**

### 1.1 Atmosphere Over Utility
Map styling configurations prioritize cinematic readability, low-light observability, psychological tone, and environmental coherence over high-density tactical GIS charts or dashboard layouts.

### 1.2 Passive Interpretation Limits
The presentation layer is structurally passive. It may smooth, stylize, or suppress elements visually, but it is strictly prohibited from fabricating runtime continuity, rewriting geographic authority, or altering telemetry states.

---

# 2. RUNTIME AUTHORITY BOUNDARIES

### 2.1 Explicit Domain Ownership
The module asserts exclusive governance over:
- Visual style sheets, layout color palettes, and global map color registries.
- Renderer appearance configurations, vector line weights, and blending opacities.
- Surface style presets and the evaluation of the single active live configuration override.
- The format and generation of immutable `MapStyleManifest` tokens.

### 2.2 Prohibited Modification Scopes
The module has **zero authority** over, and is completely blocked from mutating:
- Active position coordinates, heading vectors, telemetry data packets, or history logs in `AISRuntime`.
- Track segment buffers, decay lifecycles, or ring-buffer configurations inside `MaritimeWakeAuthority`.
- Target tracking focus assignments, scoring matrix queues, or entity tier distributions within `MaritimePopulationHierarchy`.
- Real-time visibility class determinations (`FULL`, `SILHOUETTE`, `MARKER_ONLY`, `ATMOSPHERIC_HIDDEN`) resolved by `AtmosphericReadability`.

---

# 3. INTER-SPEC CORE RELATIONSHIPS & HIERARCHIES

### 3.1 AtmosphericReadability Separation
`AtmosphericReadability` owns the authoritative assignment of a vessel's `visibilityClass`. `MapStyleAuthority` governs only the visual interpretation *within* that assigned class envelope. The style module **may never elevate a vessel to a higher visibility class** than assigned by the atmospheric engine.

### 3.2 Motion Authority Separation
Visual movement smoothing parameters operate exclusively inside screen-space frame generation loops. They are structurally blocked from modifying tracking coordinates, fixed timestep accumulation values, or underlying dead-reckoning equations inside the simulation layer.

### 3.3 Overlay Grammar Distinction
`OverlayGrammar` retains exclusive ownership over semantic composition, symbolic text layout hierarchies, and telemetry data ordering. `MapStyleAuthority` limits its tracking to the visual presentation layer components, governing traits like opacity, bloom, glow softness, and color tokens.

### 3.4 Camera Tracking Restrictions
The `ObservabilityCamera` system may observe active presentation styles for layout synchronization and frame matching only. It is strictly forbidden from parsing style registries to determine routing steps, hero targets, or cinematic pacing loops.

# 4. OVERRIDE GOVERNANCE INFRASTRUCTURE

Styling resolution follows a strict top-down cascade. Multiple concurrent override managers are prohibited to prevent execution frame conflicts.

class MapStyleAuthority {
  private activeRegistry: MapStyleRegistry;
  private maritimeRegistry: MaritimeStyleRegistry;
  private activeLiveOverride: Partial<MapStyleRegistry> | null = null;

  constructor() {
    this.activeRegistry = this.compileBaseRegistry();
    this.maritimeRegistry = this.compileBaseMaritimeRegistry();
  }

  private compileBaseRegistry(): MapStyleRegistry {
    return {
      water: { baseColor: "#080a0f", shimmerStrength: 0.15, reflectionOpacity: 0.20, currentBandAlpha: 0.10, coastlineContrast: 1.1, harborDarkness: 0.90 },
      land: { landColorHex: "#11141a", districtContrast: 0.35, coastlineVisibility: 0.60, infrastructureShadowStrength: 0.40, nighttimeDarkness: 0.85 },
      roads: { arterialOpacity: 0.30, localRoadOpacity: 0.10, glowStrength: 0.20, labelSuppression: 0.60, nighttimeFade: 0.75 },
      labels: { density: 0.40, opacity: 0.70, districtPriority: 3, infrastructurePriority: 1, suppressionStrength: 0.50 },
      atmosphere: { fogAlpha: 0.15, hazeStrength: 0.25, grainOpacity: 0.04, glowRadius: 10.0, bloomSoftness: 0.35, visibilityFalloffKm: 12.0 },
      overlays: { hudOpacity: 0.80, scannerStrength: 0.50, typographyGlow: 0.60, telemetrySoftness: 0.30, noiseSuppression: 0.50 }
    };
  }

  private compileBaseMaritimeRegistry(): MaritimeStyleRegistry {
    const defaultVessel: VesselStyle = {
      symbolic: { hullColorHex: "#3fb950", deckColorHex: "#0d1117", strokeWidthPx: 1.5, layoutScaleMultiplier: 1.0 },
      lighting: { farLightAlpha: 0.50, farLightHaloPx: 12, twinkleStrength: 0.40 },
      wakePresentation: { visualAlphaMultiplier: 0.50, edgeSoftnessScalar: 0.30 },
      motionPresentation: { headingVisualSmoothing: 0.80, interpolationCurve: "CUBIC_GLIDE", visualEasingMs: 450 }
    };

    return {
      cargo: defaultVessel,
      tanker: { ...defaultVessel, symbolic: { ...defaultVessel.symbolic, hullColorHex: "#f97583" } },
      ferry: { ...defaultVessel, symbolic: { ...defaultVessel.symbolic, hullColorHex: "#79b8ff" } },
      service: defaultVessel,
      recreational: defaultVessel,
      fishing: defaultVessel
    };
  }

  public setSingleLiveOverride(override: Partial<MapStyleRegistry>): void {
    this.activeLiveOverride = override;
  }

  public clearLiveOverride(): void {
    this.activeLiveOverride = null;
  }

  public generateManifest(simulationTimeMs: number, visibilityTokenFromAtmosphere: string): MapStyleManifest {
    let resolvedMapStyle = { ...this.activeRegistry };

    // Apply the active live override layers if registered by validation checks
    if (this.activeLiveOverride) {
      resolvedMapStyle = { ...resolvedMapStyle, ...this.activeLiveOverride };
    }

    // Inter-spec tracking containment pass: evaluate visibility thresholds under atmospheric enforcement
    if (visibilityTokenFromAtmosphere === "ATMOSPHERIC_HIDDEN") {
      resolvedMapStyle = {
        ...resolvedMapStyle,
        atmosphere: { ...resolvedMapStyle.atmosphere, fogAlpha: 1.00 }
      };
    }

    return {
      manifestId: `manifest::${simulationTimeMs}`,
      simulationTimeMs: simulationTimeMs,
      mapStyle: resolvedMapStyle,
      maritimeStyle: this.maritimeRegistry
    };
  }
}
