# Phase 1 — Harbor Population Foundation

`Goal: believable density without chaos`

## 1. Vessel Taxonomy System

This becomes the foundation for everything else.

Right now:

```
boat = generic hull
```

That must evolve into:

|Class|Role|Behavioral Identity|
|---|---|---|
|Tug|slow / directional|harbor labor|
|Ferry|route-driven|rhythmic civic motion|
|Cargo|heavy inertia|industrial gravity|
|Pilot Boat|rapid response|nervous movement|
|Coast Guard|authority|patrol behavior|
|Barge|drift-heavy|dead weight|
|Service Boat|local utility|ambient maintenance|
|Recreational|irregular|human unpredictability|

Each class eventually owns:

- dimensions
- acceleration profile
- turn rate
- wake style
- lighting
- color palette
- AIS cadence
- continuity tolerance
- route behavior
- observability weight
- cinematic importance

---

## 2. Population Layering

You do NOT want:

```
all vessels equally important
```

You want:

```
harbor hierarchy
```

### Example

|Layer|Count|Purpose|
|---|---|---|
|Hero vessels|5–20|cinematic anchors|
|Mid vessels|50–150|readable activity|
|Background vessels|300–1000|ecological density|
|Ghost traffic|1000+|atmospheric implication|

This matters because:

- readability collapses otherwise
- pacing becomes visual noise
- camera loses intentionality

---

## 3. Continuity Density

This is one of the biggest future breakthroughs.

The harbor should feel:

```
continuously alive
```

NOT:

```
constantly moving
```

Important distinction.

You want:

- lingering motion
- implied persistence
- slow state evolution
- traffic memory
- harbor rhythm

This means:

- some vessels nearly stationary
- some drifting
- some route-bound
- some arriving/departing
- some sleeping
- some fading into fog/distance

## 4. Atmospheric Readability Layer

The harbor cannot become:

```
raw AIS soup
```

Examples:

- rain amplifies wake reflections
- night emphasizes navigation lights
- distance reduces identity clarity
- density creates “port glow”
- dawn shifts industrial tone

The goal:

```
users feel the harbor before understanding it
```
