# 1. CORE WAKE DOCTRINE

The wake system preserves passive structural memory. It exists solely to catalog historical track trajectories as non-authoritative water transformations.

Wake systems may visualize:
- Path residue
- Atmospheric turbulence traces
- Directional movement memory
- Environmental readability parameters

Wake systems may NOT:
- Create continuity authority
- Extend vessel tracking truth
- Resurrect missing or dropped vessel state
- Infer vessel motion paths
- Steer the physics simulation
- Generate authoritative navigation telemetry

Vessel motion creates wake memory.
Wake memory never creates vessel motion.
