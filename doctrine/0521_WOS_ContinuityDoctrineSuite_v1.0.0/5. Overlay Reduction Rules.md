### 5.1 The Doctrine of Symbolic Subtraction

In accordance with the WOS _Symbolic Rendering_ philosophy, UI overlays (`wall/render/overlays`) do not merely decorate active data; they communicate the execution state of the world.

When a spatial primitive or district enters the _Dormant State Profile_, its corresponding UI elements must undergo **Symbolic Subtraction**. The overlay shifts from an **Expressive Grammar** (high-density tracking, active text metrics, vibrant emittance) to a **Passive Structural Grammar** (low-density, high-contrast, structural vectors). This ensures the screen remains uncluttered, low-light readable, and atmospherically calm.

Plaintext

```
[ RUNTIME STATE: ACTIVE ]               [ RUNTIME STATE: DORMANT ]
Expressive Grammar                      Passive Structural Grammar
• Vector Trail (Solid, Opacity 1.0)     • Vector Trail (Dash-Dot or Hidden)
• Rapid Telemetry Text Updates          • Frozen Coordinates / Muted Static Labels
• Blinking Signal Anchors               • Low-Opacity Fixed Radar Blip
• High Visual Density                   • "Harbor Watch" Low-Density Minimalist Grid
```

### 5.2 The Three-Tier Reduction Protocol

When the `Runtime Layer` signals a state change to dormancy, the overlay system intercepts the asset rendering and enforces three sequential modifications:

#### 5.2.1 Automatic Density Cleansing (Step 1: Thinning the Noise)

- **The Rule:** Strip all micro-telemetry data immediately upon execution suspension.
    
- **The Mechanic:** The overlay module drops transient, high-update variables from the viewport. Velocity vectors, heading readouts, sub-sampling rates, and local event frequencies are hidden. Only the absolute spatial identifiers remain: the primitive type, the persistent ID, and its geographically anchored coordinates.
    

#### 5.2.2 Grammar Commutation (Step 2: Structural Transition)

- **The Rule:** Switch from active tracking iconography to a cold, tactical infrastructure grammar.
    
- **The Mechanic:** * **Paths/Trails:** Active `Route` styles switch from `Solid` lines with high opacity to `Dash-Dot` or thin, low-opacity vectors that trace the underlying real-world map truth rather than current movement.
    
    - **Typography:** Text elements drop brightness and invert styling (e.g., changing from solid fills to hollow bounding boxes or dimmed monochrome values).
        
    - **Update Cadence:** The overlay rendering pipeline stops redrawing the asset frame-by-frame, locking its visual matrix to match the _Soft Suspend_ memory buffer.
        

#### 5.2.3 Entry into "Harbor Watch" Mode (Step 3: Atmospheric Baseline)

- **The Rule:** The asset transitions into a clean, surveillance-style telemetry baseline.
    
- **The Mechanic:** Inspired by maritime radar and public infrastructure TV feeds, the asset becomes a passive component of the terrain. It loses its individual "look at me" visual weight and inherits the global environmental settings—such as dropping opacity to blend into the _Overcast_ lighting or _Harbor Fog_ systems.

### 5.3 Surface Grammar Profiles (The Surface Connection)

Because overlays are governed by the active broadcast `Surface`, the intensity of this reduction ruleset changes depending on the channel's specific identity:

|**Surface Context**|**Active Overlay Profile**|**Dormant Overlay Behavior ("Harbor Watch")**|
|---|---|---|
|**Deep Night Drives**|Fluid path trails, soft headlamp glow matrices, active speedometers.|Trails dissolve completely. Cars leave behind a faint, fixed-opacity node on the grid layout, matching the dark background topology.|
|**Harbor Watch**|Dense AIS tracking boxes, vector arrows, real-time knots readouts.|Readouts hide. The ship icon switches to a static, hollow radar point that only pulses when a low-frequency tick registers.|
|**WOS Transit / Subway**|Moving rhythmic pulse anchors, blinking terminal schedules.|Rhythms flatten into a continuous, low-glow line marking the rail infrastructure. The train icon is absorbed by the station anchor.|

---

> 📜 **Doctrine Rule:** Overlays must never pollute the dark spaces of the map. When an infrastructure element rests, its overlay must drop its voice, strip its numbers, and sink into a quiet, tactical status indicator that respects the overarching atmosphere of the broadcast world.
