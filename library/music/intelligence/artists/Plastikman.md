# Plastikman

## Summary

Plastikman is Richie Hawtin's long-running artistic laboratory, using minimal techno as a framework for exploring systems, perception, signal behavior, and technological environments. Rather than centering songs, the project builds self-contained experiential systems.

## Why This Belongs in the Archive

Plastikman is one of the strongest StudioRich references for infrastructure-first worldbuilding. Sound, visuals, lighting, and performance all behave like components of a single operating system, demonstrating how reduction and repetition can create complete environments.
